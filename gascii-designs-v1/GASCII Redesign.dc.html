KEY CHANGES
  1. Tools move to a MacPaint-style 3×3 icon grid; active tool renders inverted (black cell, white icon).
  2. L/R mouse bindings shown as tiny corner badges on tools + one segmented control in the options bar. Right-click a tool to bind it to R.
  3. Options bar is contextual — only the active binding's options show; switching the L/R segment switches the options.
  4. Palette: bigger 26px swatches, inverted selected state, RECENT row auto-tracks last 6 glyphs.
  5. FG/BG as overlapping wells (paint-app convention) with swap (X key).
  6. Status bar keeps cell/zoom/doc; zoom gains −/+/Fit controls; selection size readout appears while a selection exists.
