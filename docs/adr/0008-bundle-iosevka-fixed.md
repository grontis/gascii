# Bundle Iosevka Fixed as the canvas font

The canvas renders exclusively with one bundled font, so its glyph coverage is the hard ceiling on what palette Pages can offer. Iosevka Fixed: near-complete coverage of box drawing, block elements, and Braille patterns (the planned future page); the Fixed variant has no ligatures (which would corrupt a character grid); OFL-licensed for bundling. Runners-up: JetBrains Mono (thinner Braille/legacy coverage), Cascadia Mono (wider cells). Coverage to be verified with a glyph torture-test sheet during scaffolding; UI chrome may use a different font freely.
